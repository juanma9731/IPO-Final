Que es?

PlantEditor es una potente herramienta para llevar cabo lavores de jardineria y agricultura, la cual nos permite desde modificar las flores de una planta, hasta permitir cambiarle la altura a nuestro arbol entre diversas funciones. Consta de un dispositivo y de una aplicacion movil la cual se puede descargar en nuestra web.

Funcionalidad

PlantaEditor permite:

1- Cambiar el tamano de un planta/arbol, esto incluye Altura, ancho del tronco y diametro de la copa.

2- Cambiar la forma de un arbusto/arbol , con distintas formas entre las que elegir.

3- Cambiar el color tanto de las flores como de las hojas con total libertad.

4- Aumentar tanto el numero de flore como de frutos de una planta.

5- Descomponer una planta

Tecnologias

La tecnologia usada en PlantEditor consta de un parte software y de un parte hardware. La parte software es una aplicacion para movil al cual nos permite llevar a cabo todas las funcionalidades explicadas. La parte hardware es un dispositivo que se coloca en la parte trasera del movil mediante un iman y se coencta a este mediante bluetooth o usb. Presenta un compartimento para introducir el componente quimico necesario para producir los cambios, asi como un compartimento de salida por el cual se dirige el componente a la planta deseada.
